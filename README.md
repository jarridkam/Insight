# Insight

Insight is a full-stack reporting application built to practice and demonstrate modern web application development.

The project currently focuses on the backend API using Java, Spring Boot, PostgreSQL, Flyway database migrations, and a clean layered architecture. A frontend client will be added later to consume the API and provide a user interface for viewing, creating, editing, and managing reports.

## Project Goal

The goal of Insight is to build a complete full-stack portfolio application that demonstrates:

- Backend API development
- RESTful API design
- Database persistence
- Database migrations
- Clean layered architecture
- Frontend integration
- Real-world project structure
- Git and GitHub workflow

Insight will eventually include both a backend API and a frontend web application.

## Current Features

- REST API built with Spring Boot
- PostgreSQL database support
- Flyway database migrations
- Report entity mapped with JPA/Hibernate
- Repository layer for database access
- Service layer for business logic
- Controller layer for HTTP endpoints
- JSON API responses
- Basic error handling for missing reports
